# List implementations

## Instruction

### Run the app:
1. Clone the repository.
2. Navigate to the project directory.
3. Run the command `deno run main.ts` to start the interactive mode.

### Tests:
1. Clone the repository.
2. Navigate to the project directory.
3. Run the command `deno test` to start the interactive mode.
